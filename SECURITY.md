# Security Policy

## Supported Versions

Currently, the following versions of **gogitopsdeployer** are supported for security updates:

| Version | Supported |
| --- | --- |
| v1.0.x | ✅ |
| < v1.0 | ❌ |

## Reporting a Vulnerability

We take the security of gogitopsdeployer seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Do not open a public issue.** Instead, please send an email to:
**enoquesousa.dev@gmail.com** (replace with real security contact if different)

### What to include in your report
- A detailed description of the vulnerability.
- Steps to reproduce the issue (PoC).
- Any potential impact if exploited.

### Our Commitment
- We will acknowledge receipt of your report within **48 hours**.
- We will provide an estimated timeline for a fix.
- We will notify you once the vulnerability is resolved.
- You will be credited in our SECURITY.md (unless you prefer to remain anonymous).
